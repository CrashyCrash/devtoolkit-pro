**Free, fast, privacy-first developer tools — all running in your browser.**

No tracking. No server calls. No ads. Just tools.

## 🛠️ Tools Included

| Tool | Description |
|------|-------------|
| **JSON Formatter** | Format, minify, and validate JSON with syntax highlighting |
| **Regex Tester** | Live regex matching with capture group display |
| **Base64 Codec** | Encode and decode Base64 text instantly |
| **JWT Decoder** | Decode JWT headers, payloads, and check token expiry |
| **Color Picker** | HSL/RGB/HEX conversion with palette generation |
| **Gradient Builder** | Visual CSS gradient creator with live preview |
| **Lorem Ipsum** | Configurable placeholder text generator |
| **UUID Generator** | Bulk v4 UUID generation with copy support |

## 🚀 Tech Stack

- **React 19** + **TypeScript 5.9** — Type-safe, modern React
- **Vite 8** — Lightning-fast builds
- **Tailwind CSS 4** — Utility-first styling with custom dark theme
- **Lucide React** — Beautiful, consistent icons
- **Code-split routes** — Each tool loads independently for fast initial load

## 📦 Development

```bash
pnpm install
pnpm dev      # Start dev server at localhost:5173
pnpm build    # Production build to dist/
```
